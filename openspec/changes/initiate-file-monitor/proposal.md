# 提案：Windows 11 文件监控系统

## 目标

实现一个具有 Win11 原生质感的文件监控系统，实时跟踪文件变更（创建、修改、删除），并提供带有高斯模糊（Mica/Acrylic）效果的高端界面。

## 背景

用户需要一个既功能强大（文件监听）又视觉出众（Win11 审美）的工具。

## 拟议变更

系统将使用 **Go** 语言和 **Wails** 框架构建。

### 核心功能

1. **高性能监控**：利用 Go 的 `fsnotify` 实现轻量级、实时的文件系统事件处理。
2. **Win11 原生审美 (Mica)**：
   - 使用 Wails 对 Windows 11 窗口背景的的原生支持。
   - UI 使用现代 Web 技术（HTML/CSS）构建，提供高级质感。
3. **单执行文件**：编译为静态二进制文件，无需额外安装运行环境。

## 设计决策

- **技术选型**：Wails。它将 Go 的高性能后端与 Web 前端的美观性完美结合。
- **窗口效果**：通过 `wails.WindowTranslucency` 开启 Mica 背景。
- **监听引擎**：`fsnotify`。Go 社区最成熟的文件监听库。

## 风险评估

- **系统兼容性**：Mica 效果仅在 Windows 11 上有效，旧版本将回退至亚克力或不透明背景。
- **性能优化**：在高频文件变更时，需要对前端事件进行节流处理。
