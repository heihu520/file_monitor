# 提案：布局自适应重构与实时状态反馈 (v8.3)

## 背景

1. **布局不自适应**：v8.2 在缩放时表现不佳，清理建议卡片拉伸变形，按钮与文字在小窗口下缺乏稳固的响应。
2. **监控感不强**：用户反馈需要更直观的方式确认“监控正在运行”，目前的界面静态感较强。

## 目标

- **引入流式网格系统**：使用 `grid-template-columns: repeat(auto-fill, ...)` 重构排行榜与清理列表。
- **新增实时监控呼吸指示器**：在实时活动页面顶部增加一个具有 `pulse` (脉冲) 呼吸特效的状态标识。
- **优化卡片原子间距**：修复复选框挤压文字，确立容器 `min-width: 0` 规范。
- **全站自适应断点**：通过媒体查询优化侧边栏与主舞台在不同视口下的比例。

## 范围

- `frontend/src/style.css`
- `frontend/src/App.vue`
