# 提案：修复 UI 滚动失效与对比度问题 (v8.1)

## 背景

v8.0 引入的多彩渐变与玻璃材质虽然美观，但在实际使用中暴露了两个问题：

1. **滚动展示范围过窄**：仅在实时活动模块的部分元素实现了滚动，且滚动逻辑在极长文件名下表现不佳。
2. **对比度不足**：粉蓝渐变在顶部区域可能由于亮度较高，导致白色的搜索框与按钮文字难以辨识。

## 目标

- **全站长文本自动滚动**：扩展滚动逻辑至“排行榜”与“活动流”的所有长文件名。
- **对比度优化**：增强顶栏背景遮罩，并为关键文字添加微阴影，确保在任何渐变底色下均清晰可见。
- **交互增强**：优化跑马灯触发机制。

## 范围

- `frontend/src/style.css`
- `frontend/src/App.vue`
