# 提案：UI 视觉美化与交互增强 (v8.0)

本项目旨在进一步提升 Monitor Pro 的视觉品质感，通过引入现代化的动效与配色方案，增强用户的交互体验。

## 背景

用户反馈当前的暗色模式较为单一，且在处理长路径/长文件名时存在遮挡或溢出问题。同时，用户希望获得更具冲击力的视觉效果。

## 目标

- **多彩背景**：实现粉蓝（Cyan-Pink）动态渐变背景。
- **滚动文案**：为过长的扫描文件名引入跑马灯（Marquee）或滚动展示效果。
- **极致模糊**：加重全局及局部卡片的 `backdrop-filter` 模糊权重，营造高级透明感。

## 范围

- `frontend/src/style.css` (样式定义)
- `frontend/src/App.vue` (模板与组件结构调整)

## 关联 Capability

- `ui`: 界面样式与交互反馈增强。
